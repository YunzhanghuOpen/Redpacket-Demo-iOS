# iOSRedPacketDemo
### Description

**iOSRedPacketDemo主要演示如何调用相关方法实现红包功能。**

分支说明：

* Master分支为红包开源版本
* jdpay 分支为红包钱包版

主要功能如下：

* 获取签名参数 
* 调用发红包页面（单聊、群聊、小额随机红包）
* 调用红包详情页面
* 调用抢红包方法
* 调用转账页面 （仅限钱包版）
* 调用零钱页面
